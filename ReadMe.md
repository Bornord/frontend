# Mon fichier pour le frontend
Le site d'Enguerran BEST, pour apprendre le frontend. 

## Set-up 

### général
 Ajouter une Vue.js en CLI.
 => npm install -g @vue/cli
(one shot)

### quotidien

Lancer vue : 
   => vue create my-first-vue-cli-app
   => aller (cd) dans le dossier qui a été créé.
   => npm run serve 

## Memento



ajouter un router
   =>  vue add router
